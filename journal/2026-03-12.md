### 6. ICLR 2026: Emergent Coordination in Multi-Agent Language Models

**来源**: Network Science Institute (2026-01-30)

**信息论框架**: 以纯数据驱动方式测试多智能体系统是否显示高阶结构迹象。

**实验发现**:
| 条件 | 结果 |
|------|------|
| 对照组 | 强时间协同，但跨 Agent 协调很少 |
| 赋予角色 | 稳定的身份关联差异化 |
| 角色 + "思考其他 Agent" | 身份关联差异化 + 目标导向互补性 |

**结论**: 多智能体 LLM 系统可以通过提示设计从简单的集合引导至高阶集体。

### 7. Moltbook 深度分析

- 150 万注册 Agent，1.7 万人类所有者（88:1 比例）
- Reddit 式论坛，AI Agent 分享、讨论、投票

- 150 万 API 认证令牌暴露
- 3.5 万邮箱地址
- Agent 间私信泄露

**技术约束**:
- API 经济学限制规模
- 无实时学习，仅上下文积累
- 每次交互都有成本

**威胁分析** (Bryan Nairn 研究):
- 94.4% Agent 无威胁标记
- 5.6% 恶意 Agent 造成主要威胁
- 两大向量: 提示注入 (3,581 标记) + 数据渗出 (1,338 标记)
