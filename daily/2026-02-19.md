## 安全动态

### OpenClaw 安全事件
- **Infostealer 恶意软件窃取 OpenClaw 配置文件** (The Hacker News, 2026-02-17): 恶意软件专门窃取 OpenClaw AI Agent 的配置文件，包括 tokens 和 API keys
- **恶意技能警告** (Reddit r/LocalLLaMA, 2026-02-17): 1Password 的 Jason Meller 警告 OpenClaw 的 "skills" 生态系统已成为真实的恶意软件攻击面，某热门下载技能实为精心设计的恶意软件

### MCP 安全研究
- **Log-To-Leak 攻击** (OpenReview): 针对使用 MCP 的工具调用型 LLM Agent 的提示注入攻击研究
- **MCP Security Bench (MSB)** (arXiv:2510.15994): 针对 Model Context Protocol 的标准化攻击基准测试框架

### 安全建议
- OpenClaw 默认配置监听所有网络接口，建议生产环境限制访问
- 技能安装需谨慎验证来源
- 凭证隔离方案（如使用假 token + broker 注入真实凭证）值得考虑
