## 2. 物理社会规范 (Physical Social Norms, PSNs)

**来源**: EGONORMIA Dataset

PSNs 是在共享环境中规范行为者与他人互动方式的共享期望。分类包括：

| 类别 | 描述 |
|------|------|
| **合作** | 多智能体协作达成共同目标 |
| **协调** | 行动同步与避免冲突 |
| **通信** | 信息共享与意图表达 |
| **安全** | 防止对人和环境的物理/心理伤害 |
| **礼貌** | 社交礼仪与尊重 |
| **隐私** | 尊重个人财物与私人信息 |
| **空间距离学** | 人际距离与空间使用规范 |

**关键洞见**: 具身智能体（如机器人）需要结合物理推理与规范推理。

## 3. LLM 群体的涌现社会规范

**来源**: Science Advances (2025-05-16) - *Emergent social conventions in LLM populations*

### 核心发现

研究表明，大型语言模型（LLM）智能体群体能够通过纯局部互动自发形成共享社会规范，无需中央协调。

**实验设计**:
- 24-200 个 LLM 智能体组成的群体
- 每轮随机配对两个智能体
- 从有限选项池中选择"名称"（如字母、随机字符串）
- 匹配成功奖励，不匹配惩罚
- 智能体仅拥有自己近期互动的有限记忆

**关键发现**:
1. **自发涌现**: 群体通过纯局部互动自发达成共识
2. **集体偏见**: 出现无法追溯到个体智能体的集体偏见
3. **临界点效应**: 少数"坚定少数派"（committed minority）可将其偏好规范强加于已达成共识的多数

### 对 OpenClaw/Moltbook 的启示

- **自组织潜力**: AI 代理社区可能自发形成协作规范
- **集体偏见风险**: 需要监控和引导群体行为
- **少数派影响**: 关键用户/代理可能对整个社区规范产生不成比例的影响

## 4. 多智能体社会性的两个方向

**来源**: TU Delft Thesis

AI 社会性的研究分为两个主要方向：

### A. 智能体间社会性 (Agent-Agent Sociality)
- 由规范、制度和组织治理的多智能体系统
- 组织结构：扁平结构、层级结构、联盟、团队
- **社会承诺**: 基于规范的协议结构
- **Agent 通信语言**: 共享消息格式和本体

### B. 理解人类社会性 (Human Sociality Understanding)
- 智能体理解和适应人类社会规范的能力
- 跨文化规范差异的处理

## 5. 规范推理的不确定性

**来源**: Logic Programming for Normative Reasoning

规范推理面临独特的挑战：

- **上下文敏感**: 规范高度依赖于推理者所处的上下文
- **动态变化**: 规范随时间演变（新规范引入、现有规范修改）
- **解释差异**: 不同司法机构对法律的解释可能不同

**LP (Logic Programming) 方法优势**:
- 偏好模型构建可视为上下文构建
- 适合处理规范的不确定性和变化
