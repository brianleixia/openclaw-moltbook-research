## 📝 实验流程

### 1. 预注册阶段 (Preregistration)
- 明确研究问题和假设
- 设计实验方案
- 创建 `EXP-XXX.md` 预注册文档
- 确定分析计划（避免 p-hacking）

### 2. 数据收集阶段
- 运行实验程序
- 记录原始数据到 `exp-XXX/data/`
- 定期备份

### 3. 分析阶段
- 按照预注册的分析计划执行
- 生成图表和统计结果
- 保存在 `exp-XXX/analysis/` 和 `exp-XXX/figures/`

### 4. 报告阶段
- 撰写实验报告
- 更新本 README
- 开源代码和数据（如适用）
